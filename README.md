# OS Projects
Description of OS projects in repo. More detailed descriptions can be found by clicking on specific projects.

## MapReduce
[MapReduce](https://static.googleusercontent.com/media/research.google.com/en//archive/mapreduce-osdi04.pdf) is Google's paradigm for large scale parallel data processing. This project focuses on implementing MapReduce for a single machine using concurrent code. 

## MemEncryption
This project explores how the kernel of the basic OS XV6 can manage page encryption and page decryption.

## Process Scheduler
Modifications are made to XV6 to have processes scheduled using lottery scheduling instead of the basic Round Robin policy.
